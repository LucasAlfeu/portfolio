# Como executar

## Pré-requisitos: npm/yarn

```bash
# clonar repositório
git clone https://github.com/LucasAlfeu/portfolio.git

# entrar na pasta portifolio
cd portifolio

# instalar as depedências
npm install

# iniciar o projeto
npm start
```
